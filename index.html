Metimeter: an AI/ML DSS Quality Assessment tool
The metimeter (from metis, skill, and meter) is an online calculator aimed at helping serious pratictioners in evaluating AI-based decision support systems by examining key areas like data reliability, model robustness, calibration, utility, and AI's effect on human decisions.

The metimeter features a structured six-step method, yet users may freely select and use any particular tool below according to their needs.
TOOLS
  • Data Reliability
  • Data Similarity
  • Model Calibration
  • Model Utility
  • Model Robustness
  • Human Interaction
